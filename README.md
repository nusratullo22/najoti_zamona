# Наҷоти Замона (Najoti Zamona)

Ин лоиҳа як вебсайти муосир барои муаррифӣ, шиносоӣ ва дастрас намудани доруҳои гиёҳӣ ва табобатии табиӣ мебошад. Мақсади асосии сайт кумак ба одамон барои пайдо кардани роҳҳои табиии табобат ва тарғиби тарзи ҳаёти солим аст.

---

## 🌟 Хусусиятҳои асосии сайт
* **Каталоги доруҳо:** Рӯйхати пурраи гиёҳҳои шифобахш бо тавсифи муфассал.
* **Ҷустуҷӯи осон:** Имконияти зуд пайдо кардани гиёҳҳо барои бемориҳои муайян.
* **Маълумоти муфид:** Мақолаҳо ва тавсияҳо дар бораи тарзи дурусти истифодаи гиёҳҳо.

---

## 🛠️ Тезнологияҳо (Technologies Used)
Барои сохтани ин сайт технологияҳои зерин истифода мешаванд:
* **HTML5** - барои сохтори сайт.
* **CSS3** - барои дизайни зебо ва намуди сайт.
* **JavaScript** - барои қисмҳои динамикӣ ва интерактивӣ.
